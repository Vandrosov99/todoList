Если первый раз
1) install node (node -v , npm - v)
2) npm init 
3) npm i gulp-cli -g
4) npm install --save-dev gulp gulp-sass browser-sync
5) gulp style(один раз запуск для инициализации файлов)
6) gulp watch.

Если > 1 
1)npm init
2) npm install --save-dev gulp gulp-sass browser-sync
3) gulp style(один раз запуск для инициализации файлов)
4) gulp watch.
